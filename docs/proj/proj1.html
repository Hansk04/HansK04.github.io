Soccer Video Analysis with Roboflow

Core Idea

View on GitHub

This project explores how data science can deepen our understanding of football tactics and performance.

Using detailed player tracking data and event logs, I built an expected goals (xG) model to estimate the likelihood of each shot resulting in a goal. The project covers:

  • Data cleaning & feature engineering
  • Exploratory data analysis to reveal team and player trends
  • Training machine learning models (logistic regression & tree-based) to predict xG
  • Visualizations to explain spatial patterns and shot quality
